# CRM

## Prerrequisitos
- Python 2.7.13
- virtualenv
- python-pip

## Configuración
```
$ virtualenv crm
$ source crm/bin/activate
```

## Instalación
### Dependencias
```
$ pip install django
$ pip install gmail
$ pip install hashids
$ pip install dj_static

```

## Ejucutar
```
python manage.py runserver
```
